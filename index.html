aevum — live AI music performance, for macOS

A macOS app for live AI music performance, built on Magenta RealTime 2. Load songs, auto-extract musical loops, morph between them in real time, and control every generation parameter live with a MIDI controller.

№ 02 Open Source

cocotagger — semantic music tagging, zero-shot

Tag audio with genre, mood, instruments, vocals, and texture — zero-shot via 768-dimensional MusicCoCa cosine similarity. 178 tags across 10 categories; runs entirely on a MacBook CPU, no GPU required.

stack
Python · TFLite · XNNPACK
model
MuLan → CoCa (MusicCoCa) · 44M YouTube tracks
tags
178 across genre / mood / instruments / vocals / texture / harmonic / tempo / era / production / energy
